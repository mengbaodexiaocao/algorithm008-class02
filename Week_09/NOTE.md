# 学习笔记

## 动态规划模板  

```
function DP() {
  const dp = [M][N];
  for (let i = 0; i < M; i++) {
    for (let j = 0; j < N; j ++) {
      dp[i][j] = _Function(dp[i'][j']);
    }
  }
}

```

- 动态规划 和 （递归、分治）没有根本上的区别 （关键看有无最优子结构）  
- 共性：找到重复子问题  
- 差异性：最优子结构、中途可以淘汰次优解  


## 不同路径2状态转移方程   
```
dp[i][j] = dp[i-1][j] + dp[i][j-1]
```

## 高级DP  
- 状态拥有更多的维度 （二维、三维甚至更多）  
- 状态方程更加复杂
